# ‚≠ê [Hexlet](https://ru.hexlet.io/my) Frontend Final Project
## "Slack" Chat

### Features:
- user auth (login, logout, signup)
- send/receive messages
- add, rename, remove channels
- profanity filter
- change language, supported locales: ru (default), en

### Technologies:
